# 📱 Répertoire de contact professionel pour la recherche d'alternance

Une application Windows Forms C# (.NET Framework 4.7.2) rudimentaire permettant d'enregistrer simplement et intuitivement les coordonnées d'entreprises pour ma recherche d'alternance.

## 🚀 Fonctionnalités
- Interface graphique intuitive et opérationnelle.
- Possibilité d'enregistrer un nombre infini de contact.
- Recherchez et triez les entreprises enregistrées dans votre liste de contacts selon les groupes "Contactée" et "Non contactée" ou selon le groupe "Tout" pour afficher tous vos contacts.
- Prise en charge d'un bouton "Ajouter" qui permet l'ouverture d'une fenêtre dédiée afin de saisir les coordonnées des contacts à enregistrer.
- Prise en charge d'un bouton "Supprimer" pour retirer un contact de la liste de contacts.

## 🛠️ Technologies utilisées
- **Langage :** C# - Windows Forms (.NET Framework - 4.7.2)
- **IDE :** Visual Studio Community 2026
- **OS :** Windows

## 📦 Installation et Lancement
1. Cloner le dépôt : `git clone https://github.com/lucasdelennepro-dev/Repertoire-Csharp.git`
2. Ouvrir **Visual Studio 2026**.
3. Faire `File` > `Open` > `Project/Solution` > `Existing Projects into Workspace (file .slnx)`.
4. Sélectionner le dossier du projet.
5. Lancer le fichier `Repertoire.slnx`.
